TimeWaster
==========
Just for fun!

###### How to run on OS X
```
curl -o TimeWaster.class https://raw.githubusercontent.com/Koenkk/TimeWaster/master/TimeWaster.class
java TimeWaster
```

###### How to run on Linux
```
wget https://raw.githubusercontent.com/Koenkk/TimeWaster/master/TimeWaster.class
java TimeWaster
```
